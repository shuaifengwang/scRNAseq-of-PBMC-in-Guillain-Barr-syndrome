# scRNAseq-of-PBMC-in-Guillain-Barr-syndrome
The codes were associated with quality control and data visualization

## cellranger.sh 
This file was the code which our team used Cell Ranger(version: 7.2.0) software to generate expression matrix.

## soupx.md
The codes in this file were used to filter the background RNA pollution.

## DoubleFinder.md
The codes in this file were used to filter the double cells in one oil drop.

## cellchat.md
The codes in this file were used to analyses the cells contract which we intersted.

## monocle3.md
The codes in this file were used to analyses the developmental trajectory of cell subpopulations, i.e. the chronological order of cell development.

## Data integration, cluster analysis, cell annotation.md
The codes in this file  included samples intergartation, dimensionality reduction clustering, marker gene analysis, and cell annotation. 
Each steps were written in great detail, someone who use this codes could easily repeat our jobs.

## figure.md
The codes in this file were used to generate images.

## sessionInfo.md
This file was the environmental configuration information for R script.
